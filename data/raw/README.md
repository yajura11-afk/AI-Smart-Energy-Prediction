Dataset Name: Energy Consumption Dataset
https://github.com/rskworld/energy-consumption/blob/main/energy_consumption.csv 

Number of Samples and Features in the dataset: 43,800 samples and 6 original features (timestamp, household_id, consumption_kwh, temperature, hour, day_of_week). 
After feature engineering, additional features (year, month, day, minute) were extracted from the timestamp.
