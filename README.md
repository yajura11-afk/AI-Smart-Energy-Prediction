# AI-Powered Smart Energy Consumption Prediction

## Project Overview

Predicts household energy usage using Machine Learning and
provides AI-generated recommendations to reduce electricity
consumption.

## Features

- Data preprocessing
- Energy prediction
- Interactive dashboard
- AI recommendations

## Technologies

- Python
- Pandas
- Scikit-learn
- Streamlit
- OpenAI API
  
## Dataset
https://github.com/rskworld/energy-consumption/blob/main/energy_consumption.csv
